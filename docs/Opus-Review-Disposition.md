# Opus and Fable Review Disposition

This document preserves the review-to-build trace without publishing the private build history. All Opus blockers and high-priority findings were accepted into the revised plan; Fable then closed the remaining seams through binding amendments A1–A13.

## Opus blockers

| Finding | Disposition | Release 1.0 implementation |
|---|---|---|
| B1 — Distribution exempt from governance | Accepted | Validated repository improvement proposal and loop contract, named owner, budgets, review clocks, kill and conditional supersession |
| B2 — 1.0 unreviewable | Accepted | Two shipped packs; eight inert declarations; bounded base/runtime; platform and corpus payload deferred |
| B3 — No Git-history scan | Accepted | Fresh one-commit export plus full-history publication scan |
| B4 — Unfalsifiable completeness | Accepted | Closed 475-file public denominator and 12-family private/local denominator |
| B5 — Unbounded registry context | Accepted | Three routing levels, Level 0 ≤400, category cap 20, operational core ≤3,000 |
| B6 — Evaluations lack controls/actions | Accepted | Paired arms, three trials, hidden rubrics, deterministic trace assertions, fixed delta threshold |
