## Conditional supersession

This repository remains an experiment until all of these conditions pass:

1. every 1.0 mechanical and human release gate passes;
2. a non-builder completes and records a clean-room install;
3. at least five real uses are recorded during a blocker-free 30-day window;
4. the operator approves predecessor deprecation;
5. each predecessor README is edited to read-only/superseded status and points here.

Operator self-use counts toward usefulness clocks but not external-demand triggers. Until the conditions pass, the five sources remain unchanged, authoritative at their pins for provenance, and not represented as deprecated.
