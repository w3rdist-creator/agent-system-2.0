# Operator Correction Rule

Append every material operator correction with its evidence handle, prior hypothesis, and resulting hypothesis update. One event changes the local hypothesis; it does not silently rewrite standing doctrine.

Standing doctrine changes only after repeated evidence reaches the owner-declared threshold or the operator gives one explicit strong instruction within their authority. Record the before and after text, change owner, review date, and superseded event. Preserve rejected or superseded corrections for auditability.

Conflicting corrections remain separate until scope, freshness, or authority resolves them. Never infer a durable preference from silence.
