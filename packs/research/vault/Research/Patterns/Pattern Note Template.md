# Pattern — NAME

## Observations in two independently sourced domains

State each observation, population, time scale, constraints, and source limit separately.

## Explicit causal mechanism

Map inputs, actors, constraint or incentive, transformation, output, and feedback in neutral language; label any link supported only by analogy.

## Falsifiable prediction

Name an observable target-domain result and review horizon.

## Strongest objection and rival explanation

Preserve the strongest live alternative.

## Strategic-interaction and Goodhart check

When adapting actors are present, record incentives, signals, repeated interaction, likely equilibrium, proxy metric, Goodhart pressure, failure mode, and countermeasure. Otherwise record `not-triggered` and why.

## Creative-to-empirical firewall

State whether any source is creative, metaphorical, speculative, or historical; require independent target-domain contact and a counterexample before factual promotion.

## Constructive counterweight

Name a supported constructive mechanism or explicitly state why search found none. Do not invent symmetry.

## Non-applicability

State where a required causal link is absent. If the bridge fails, set status `non_connection` and stop promotion.

## Provenance

Name stable source-row coordinates and local source notes.

## Disposition and decision delta

Use `candidate_connection | supported_connection | non_connection | merged | demoted | rejected`; state the concrete artifact, question, experiment, or decision changed.
