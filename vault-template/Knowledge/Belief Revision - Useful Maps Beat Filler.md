# Belief Revision — Useful Maps Beat Filler

## Prior state

2026-07-10, confidence 0.60: “no empty advertised layer” might require several topical content notes in every folder.

## New evidence

The binding brief defines the minimum as a landing page in every advertised folder and separately forbids filler. [[Reviews/Review - Phase 4 Seed Scope]] confirmed that broad topical seeding would invent consumer needs.

## Revision event

2026-07-10: narrowed the implementation rule to one useful local map per layer and content only for a real source-grounded chain.

## Current state

Confidence 0.90: local maps establish honest navigation; topical notes require provenance and a real consumer. Falsifier: a clean-room user cannot locate where a legitimate artifact belongs using the maps.

## Decision delta

Sixteen local maps ship; invented domain notes do not. This changed the Phase 4 artifact set and future extraction rule.
