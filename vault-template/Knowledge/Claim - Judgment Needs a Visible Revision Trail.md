# Claim — Judgment Needs a Visible Revision Trail

Grounded claim: a system claiming to improve judgment should preserve the source, the prior belief, the evidence that changed it, and the revised state. Otherwise later correctness can be hindsight reconstruction rather than evidence of calibration.

Provenance route: [[Sources/Source Note - Truth-Seeking Engine]]. Strongest objection: recording revisions creates overhead and does not itself improve judgment. Falsifier: if the record never changes a decision, extraction rule, or future review, it should be demoted as ceremony.

Application value: test the claim inside the actual Phase 4 vault build through [[Knowledge/Mechanism - Append Before Canon]].
