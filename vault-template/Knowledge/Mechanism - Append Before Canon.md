# Mechanism — Append Before Canon

Mechanism: preserving prior state before revision makes the evidence-to-belief transition inspectable. Requiring an objection, falsifier, and decision delta then makes promotion costly enough to filter attractive but unused claims.

This is a mechanism, not proof of outcome. Non-connection boundary: append-only logs can preserve history without improving decisions when no review consumes them. Demote this candidate if the Phase 4 chain produces no changed artifact or rule.
