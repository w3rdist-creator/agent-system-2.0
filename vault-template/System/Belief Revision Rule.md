# Belief Revision Rule

Corrections append dated events. Preserve the prior belief, confidence, scope, new evidence, authority limits, revised state, falsifier, and `decision_delta`. Never rewrite history to make an earlier judgment look calibrated.

One ordinary feedback event updates a hypothesis. Standing doctrine changes after repeated evidence or one explicit strong instruction. The seed example is [[Knowledge/Belief Revision - Useful Maps Beat Filler]].
